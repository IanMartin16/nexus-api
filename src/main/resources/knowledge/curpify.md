# Curpify

Curpify es una API del ecosistema Evilink enfocada en validación estructurada en México.

## Capacidades principales

### Validación de CURP
Curpify permite validar CURP y trabajar con datos estructurados relacionados.

Casos de uso:
- validación de identidad basada en CURP
- flujos de registro
- alta de usuarios
- verificación básica de datos estructurados en México

### Validación de RFC
Curpify también permite validar RFC y CURP con dígito verificador.

Casos de uso:
- validación fiscal estructurada
- revisión de datos fiscales básicos
- flujos donde se requiere validar RFC de forma programática

## Enfoque actual

Curpify está orientado a:
- validación de CURP
- validación de RFC con dígito verificador
- flujos técnicos de identidad y validación fiscal estructurada en México

## Lo que Curpify no es

Curpify no debe presentarse como:
- una suite completa de KYC
- una solución de biometría
- una plataforma de validación documental general
- una plataforma de identidad global

Su foco actual es validación estructurada de CURP y RFC vía API.

## Integración

Curpify está diseñado para integrarse fácilmente en sistemas y servicios que necesiten validar datos mexicanos de identidad o fiscales de forma programática.
