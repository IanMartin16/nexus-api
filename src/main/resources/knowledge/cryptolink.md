# CryptoLink

CryptoLink es una API del ecosistema Evilink orientada a datos financieros y cripto, con capacidades de consulta, señales y lectura resumida de mercado.

## Capacidades principales

### Price Lookup
Permite consultar precios de activos soportados.

Casos de uso:
- mostrar precios en aplicaciones
- alimentar dashboards
- integrar datos de mercado en tiempo real o con latencia controlada

### Market Snapshot
Permite obtener una lectura resumida del mercado con KPIs principales.

Casos de uso:
- vistas generales de mercado
- widgets de resumen
- contexto rápido para usuarios o sistemas

### Trends
Resume dirección de mercado y movimiento reciente.

Casos de uso:
- lectura rápida de tendencia
- contexto para análisis técnico ligero
- señales de comportamiento reciente

### Momentum
Evalúa fuerza y consistencia del movimiento reciente.

Casos de uso:
- lectura de tracción de activos
- identificación de fuerza o debilidad relativa
- apoyo a análisis resumido

### Risk Flags
Señala alertas relevantes de riesgo en activos soportados.

Casos de uso:
- monitoreo de riesgo
- detección de condiciones sensibles
- soporte a lectura de mercado más prudente

### Anomalies
Detecta comportamientos o movimientos atípicos.

Casos de uso:
- identificar outliers
- revisar actividad inusual
- complementar flujos de monitoreo

### Market Health
Resume el estado general del mercado.

Casos de uso:
- lectura agregada de condiciones de mercado
- contexto general para sistemas o usuarios

### Social Pulse
Resume narrativa y pulso social del mercado.

Casos de uso:
- observación de narrativa dominante
- complemento a señales de mercado
- contexto adicional para activos cripto

## Enfoque actual

CryptoLink está orientado a:
- consumo de datos financieros y cripto vía API
- señales resumidas de mercado
- soporte a integraciones, dashboards y flujos analíticos ligeros

## Lo que CryptoLink no es

CryptoLink no debe presentarse como:
- una plataforma completa de trading
- un exchange
- una suite institucional de análisis financiero general
- una plataforma de ejecución de operaciones

Su foco actual es proveer datos, señales y herramientas de lectura de mercado dentro del ecosistema Evilink.

## Integración

CryptoLink está diseñado para integrarse en aplicaciones, dashboards, servicios y flujos técnicos que requieran datos de mercado, señales y contexto financiero/cripto.
